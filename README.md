# 🤖 Claude Demo Streamlit App

A demo Streamlit app built and deployed to GitHub entirely by [Claude](https://claude.ai).

## Features
- 📊 Dashboard with live metrics and charts
- 📈 Interactive chart explorer (line, scatter, histogram, pie)
- 🎲 Random data generator with correlation heatmap
- ℹ️ About page

## Deploy to Streamlit Cloud (free)

1. Fork this repo
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Sign in with GitHub → New app → select this repo → `app.py` → Deploy

Live in ~60 seconds. No credit card needed.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
